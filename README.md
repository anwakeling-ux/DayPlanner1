# Day Planner

A simple daily planner app for Android with hourly timeline and alarm notifications.

## Features
- 24-hour daily timeline
- Add events with title, notes, time, and color
- Alarm notifications when events start
- Navigate between days
- Tap to edit, long-press to delete

## Building
This project is set up with GitHub Actions to build automatically.
Push to GitHub and the APK will be built for you — see the Actions tab.
